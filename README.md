PhotosFX is a Java-based Android application designed for browsing, organizing, and managing photo collections through an intuitive mobile interface. The project emphasizes clean object-oriented design, responsive UI behavior, and practical application of core software engineering principles.



Features

    Album creation and management for organizing photo collections
    
    Photo tagging by person and location
    
    Advanced search functionality supporting AND / OR logic with case-insensitive matching
    
    Full-screen slideshow mode for immersive photo viewing
    
    Real-time UI updates when adding, removing, or modifying albums and photos

Technical Highlights

      Built using Java with a focus on object-oriented programming principles
      
      Modular design for managing photo metadata and application state
      
      Persistent storage to maintain albums and tags across sessions
      
      User-friendly Android GUI designed for efficient navigation and interaction

Learning Objectives

      This project was developed to strengthen understanding of:
      
      Object-oriented design and data modeling
      
      Mobile UI development and state management
      
      Search logic implementation and string handling
      
      Building maintainable, extensible Android applications
